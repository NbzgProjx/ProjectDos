ProjectDos

Educational Python dos project.

Features

- Termux
- Multi-threaded
- Famous Dos Tools implented
- Auto git clone support

---

Installation

Termux

pkg update && pkg upgrade -y
pkg install python git clang -y

---

Clone Repository

git clone https://github.com/NbzgProjx/ProjectDos.git
cd ProjectDos

---

Requirements

Install Python requirements:

pip install -r requirements.txt

---

Run

python3 project_dos
